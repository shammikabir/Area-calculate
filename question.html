What is DOM and what is the purpose of DOM

Ans-DOM represents Documents Object model.The Document object model gives JavaScript all the functionality it needs to create dynamic HTML.Using DOM, JavaScript can change any HTML element on the page, HTML attributes ,all CSS styles,can remove existing HTML elements and attributes,can add new HTML elements and attributes.

What is event bubble?

Ans-Event bubbling is a method of propagating events in the HTML DOM. when an event is on an element within another element and both elements have registered a handle to that event. This is a process that starts with the element that raised the event and propagates through the elements involved in the hierarchy. In event bubbling, events are first captured and handled by the innermost element and then propagated to the outer elements.

What is a callback function and why will you use it?

Ans-A callback function is a function that is passed as an argument to another function and then called within the outer function to complete some action. Use callback functions in JavaScript when you want a function to execute only after another function has finished executing. To create a function callback you need to pass it as a parameter to other functions.
